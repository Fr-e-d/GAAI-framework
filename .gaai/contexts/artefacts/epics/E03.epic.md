# Epic E03: Prospect Gets Qualified and Matched to the Right Expert

## Purpose

A company with a real AI automation need can describe their project, see AI-extracted requirements confirmed, and receive a ranked list of matched experts with transparent per-criterion scores — so they can make an informed choice without wading through irrelevant profiles.

> **Layer constraint:** This epic delivers Layer 3.2 UI only (callibrate.ai + satellite sites). All extraction, scoring, and matching logic is Layer 2 — already built in E06S07, E06S08, E06S05. E03 consumes those APIs and renders the results.

---

## Scope

- Freetext project description input (calls `POST /api/extract` from E06S08)
- Confirmation step UI: displays low-confidence fields as targeted questions (max 2–3), skipped if all fields high-confidence
- Anonymized match results display: top-3 experts, global score + per-criterion breakdown (quality_tier badge)
- Directory mode: browse expert profiles, filter by vertical/skills, ranked by composite_score
- Prospect selects an expert to book a call (triggers E04)

---

## Out of Scope

- AI extraction logic (Layer 2 — E06S08)
- Matching computation (Layer 2 — E06S05, E06S07)
- Prospect account creation (anonymous at MVP)
- Prospect payment (prospects use the platform free)
- Saved prospect searches or history

---

## Stories

- E03S01: Prospect can describe their project in freetext and submit (freetext input + calls extraction API)
- E03S02: Prospect sees AI-extracted requirements and can confirm low-confidence fields (UI confirmation step)
- E03S03: Prospect sees a ranked list of anonymized matched experts with transparent scores and quality tier
- E03S04: Prospect can browse the expert directory (directory mode) and select an expert to book

---

## Success Metrics

- Freetext submission rate ≥60% (started → submitted)
- Confirmation step completion rate ≥80% (of those who reach it)
- Prospects rate expert list relevance ≥7/10
- Average time submission → matched list displayed <5 seconds

---

## Dependencies

- E02 (Expert profiles must exist before matching is possible)
- E06 (E06S07 satellite funnel API + E06S08 extraction + E06S05 matching must be active)
